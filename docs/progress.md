# 개발 현황

기능별 Backend/Frontend 구현 상태를 추적하는 문서다.
MVP 범위는 `commerce-backend/docs/PRD.md` 기준이다.

## MVP 기능

| 기능 | Backend | Frontend |
|---|---|---|
| 회원가입 / 로그인 / 토큰 재발급 | ✅ | ⬜ |
| 상품 목록 / 상세 조회 | ✅ | ⬜ |
| 관리자 상품 관리 (등록/수정/삭제) | ✅ | ⬜ |
| 관리자 재고 관리 (생성/조정/이력) | ✅ | ⬜ |
| 주문 생성 / 취소 | ✅ | ⬜ |
| 결제 준비 / 승인 (네이버페이) | ✅ | ⬜ |

## MVP 제외 항목

배송, 쿠폰/프로모션, 장바구니, 리뷰, 정산/운영 백오피스, 다중 PG 연동은 MVP 범위 밖이다.

- Frontend 기능 완료: ✅ 표시
